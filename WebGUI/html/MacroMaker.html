Instructions

To read or write:
1. Select one or more FEs from the list of FEs.
2. Enter the address to read from/write to. Data is also required for a write.
3. Select the format of your address and data (hex/dec/ascii).
4. Select either "Write" or "Read" to finish your manual command.
