Making Changes

Add/Save/Delete are all local operations. Nothing is saved to the server until 'Submit Changes to Server' is clicked. So be sure to click 'Submit Changes to Server' when you are done modifying the icon list.

Permissions

Permissions Note: 0 := Inactive, 10 := Standard User, 100 := Expert, 255 := Super User

Images

Place icon images in: ots/otsdaq-utilities/WebGUI/images/iconImages/
...and only enter image filename in form.

If you enter 0 then the Alt field will be used as the icon.

Webpages

Place icon window webpages in sub-directories here: ots/otsdaq-utilities/WebGUI/
...and enter path to page as /WebPath/<path to filename>
